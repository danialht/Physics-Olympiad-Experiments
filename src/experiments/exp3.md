## C++ Implementation

## Wave Equations
The fundamental wave equation is:

$$\frac{\partial^2 u}{\partial t^2} = c^2 \nabla^2 u$$

Where:
- **u** is the wave function
- **c** is the wave velocity
- **∇²** is the Laplacian operator

## Simulation Parameters
- Time steps: 1000
- Grid resolution: 100×100
- Time step size: 0.01 s
